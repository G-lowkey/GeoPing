📍 GeoPing

Built out of frustration. Shared for everyone going through the same thing.


The Problem
You're heading out. Your parents want to know where you are. So you open WhatsApp, find your location, send it manually — every single time.
It's a small thing. But it adds up. And it's annoying.
GeoPing fixes that.

What is GeoPing?
GeoPing is a free, open-source Android app that automatically shares your location with your chosen contacts on a schedule — no manual sending, no forgetting, no friction.
Set it once. It handles the rest.

Features (Planned)

📅 Scheduled location sharing — set times for automatic sharing
👨‍👩‍👧 Custom contacts — choose exactly who gets your location
📲 WhatsApp/SMS integration — no new app needed for your family
🔔 Smart notifications — know when sharing starts and stops
🔒 Privacy first — you're always in control


Why I Built This
I got tired of manually sending my location to my parents every time I went out. I figured if this annoyed me, it probably annoys a lot of people — especially in families where location sharing is just part of daily life.
So I decided to build a solution. And document the whole journey — from zero coding knowledge to a published Android app.
